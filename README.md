# TRR-project

## Traffic Lights Module

###### Programming language used  
**Python**
###### Major Component
**Raspberry Pi 3B+**
###### Hardware Components
* LEDs
* Buzzer
* 1-digit 7-segment display
* T-GPIO Extension Board
* 220Ω resistors 

## Floor Cleaner Module

###### Programming language used
 **MicroPython**
###### Major Component
**Raspberry Pi Pico**
###### Hardware Components
* Motors
* Servo motor
* NPN transistor
* Ultrasonic distance sensor
* Water pump 
* L293D motor driver
* Battery 
* caster ball

## MQ-3 Alcohol Detection Module

###### Programming language used
**Python**
###### Major Component
**Raspberry Pi 3B+**
###### Hardware Components
* MQ-03 Alcohol Sensor
* Motor
* NPN Transistor
* Relay
* 220Ω resistor 
* 1N4007 Diode
